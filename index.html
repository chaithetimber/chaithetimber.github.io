Drivetrain Differentials (How turning works)

A differential describes how left and right wheels can rotate at different speeds, allowing smooth turning. In VEX V5, this is usually done by software control rather than a physical gear differential.

Instead of a mechanical differential, VEX robots simulate this by adjusting motor speeds independently.

Odometry Wheels (Tracking Position)

Odometry wheels are passive tracking wheels that measure robot movement using encoders. They allow accurate autonomous positioning.

Most advanced systems use 2–3 tracking wheels to measure X, Y, and rotation.

Control System Code (C++)


#include "vex.h"

using namespace vex;

competition Competition;

motor leftMotor(PORT1, ratio18_1, false);
motor rightMotor(PORT10, ratio18_1, true);

int main() {
    vexcodeInit();

    while (true) {
        int forward = Controller1.Axis3.position();
        int turn = Controller1.Axis1.position();

        leftMotor.spin(forward + turn);
        rightMotor.spin(forward - turn);

        wait(20, msec);
    }
}
